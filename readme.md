# ВНИМАНИЕ!!!
- Сборка предназначена для Project Zomboid Build 42
- Перед установкой модпака сохраните ВСЕ ваши оригинальные файлы, на случай, если что-то пойдет не так и вы могли восстановить свою игру без потерь
# Как установить:
1. Подписаться на [коллекцию модов](https://steamcommunity.com/sharedfiles/filedetails/?id=3600514510) (можно добавить к своим не заменяя)
2. Распаковать папку `ProjectZomboid` в папку `common` по пути `..\\steamapps\common\ProjectZomboid` с заменой файлов (установка [Realistic Car Physics [B42]](https://steamcommunity.com/sharedfiles/filedetails/?id=3559765660))
3. Распаковать папку `Zomboid` в папку `Документы` по пути .`.\\%user%\Zomboid` с заменой файлов (мой мод на некоторые переводы + настройки модов)
4. Зайти в игру
5. Зайди в меню модов
6. Снизу слева активировать пресет `lxckxr's_modlist`
7. Нажать "Принять"
8. По желанию можно отключить любые из переводов (используются только моды от [Malevi4](https://steamcommunity.com/id/iMalevi4/myworkshopfiles/?appid=108600) + мой ручной перевод некоторых непереведенных модов)

# Installation Instructions:
1. Subscribe to the [mod collection](https://steamcommunity.com/sharedfiles/filedetails/?id=3600514510) (you can add it without replacing your existing mods).
2. Extract the `ProjectZomboid` folder into `..\\steamapps\\common\\ProjectZomboid`, replacing files if prompted (includes [Realistic Car Physics [B42]](https://steamcommunity.com/sharedfiles/filedetails/?id=3559765660)).
3. Extract the `Zomboid` folder into `..\\%user%\\Zomboid`, replacing files if prompted (contains my translations and mod settings).
4. Launch the game.
5. Go to the Mods menu.
6. In the bottom-left corner, activate the preset `lxckxr's_modlist`.
7. Click "Accept".
8. Optionally, disable any translations (only mods from [Malevi4](https://steamcommunity.com/id/iMalevi4/myworkshopfiles/?appid=108600) are included, plus my manual translations for some untranslated mods).
